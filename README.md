This project performs Topic Modeling analysis on national and international news media coverage related to Universitas Airlangga. The objective is to uncover dominant themes, compare narrative patterns, and analyze how different media sources frame similar topics.

Latent Dirichlet Allocation (LDA) is applied to extract hidden thematic structures from news articles written in Indonesian and English.
